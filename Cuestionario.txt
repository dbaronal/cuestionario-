3. ¿Cómo se organiza internet? Habla de las direcciones IP y los DNS.
R: El DNS (Sistema de Nombres de Dominio) es el encargado de “traducir” los números de una dirección IP en un nombre de dominio. Es decir que convierten el dominio escrito en el navegador y dirigen el acceso hacia el servidor al que apunta el dominio.

R:HTTP es un protocolo de acceso para las páginas web a través de Internet. HTTP son las siglas para Hypertext Transfer Protocol que se traduce al español como el “protocolo de transferencia de hipertextos”. ... El HTML es el lenguaje utilizado para codificar la información en los recursos web.
